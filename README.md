 Mutli-speak AI
A multilingual chatbot built with **Python** and **Streamlit** that supports **English, Hindi, Telugu, and German**. It automatically detects the user's language, translates messages, performs sentiment analysis, and answers questions using a dataset with partial matching.

 Features
- Multilingual support
- Roman Hindi & Roman Telugu support
- Automatic language detection
- Sentiment analysis
- Dataset-based Q&A
- Streamlit user interface

 Tech Stack
- Python
- Streamlit
- Pandas
- LangDetect
- Deep Translator
- Scikit-learn

 Run the Project

bash
pip install -r requirements.txt
streamlit run app.py
